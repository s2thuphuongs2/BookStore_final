# Chạy Book Store

## B1 - Chuẩn bị ứng dụng

Chuẩn bị Intelij IDEA và Docker

## B2 - Mở project bằng Intelij IDEA

1/ Chạy File docker-compose.yml

2/ Kết nối và chạy file bookstore.sql

******Database:****** bookstore

**User**: root

**Password**: <để trống>
